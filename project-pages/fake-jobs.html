Detecting Fake Job Listings
Built a semi-supervised ML system to detect fraudulent job ads using structured features, embeddings, and real-world intuition.
Project Details / Background
This project started from a simple but important question: how can we tell if a job listing is real or fake? I gathered data from LinkedIn, Indeed, and a Kaggle dataset, and built a semi-supervised learning pipeline that combined structured features with large language models to uncover patterns that human readers might miss.
I experimented with different feature engineering approaches (including document length, salary mentions, keyword usage) and used pseudo-labeling to improve the modelâ€™s ability to generalize. Ultimately, the project highlighted how a mix of intuition, automation, and clean data can come together to solve a real-world challenge with real impact.
